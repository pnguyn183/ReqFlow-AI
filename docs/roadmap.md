# Delivery Roadmap

| Sprint | Scope | Exit criteria |
| --- | --- | --- |
| 1 | Authentication, RBAC, requirement repository | Users securely create, search and import requirements |
| 2 | Requirement analysis | Classification, extraction, stakeholders and quality gaps are reviewable |
| 3 | User story generation and prioritization | Stories, acceptance criteria, DoD, MoSCoW and RICE are operational |
| 4 | Documentation center | BRD, SRS and backlog export to PDF/DOCX |
| 5 | Process modeling | Use case, BPMN and ERD suggestions are generated |
| 6 | Traceability and change control | Coverage matrix, version history and approval workflow are complete |
| 7 | AI assistant and vector retrieval | OpenAI Q&A, missing requirements and semantic context retrieval |
| 8 | Hardening and deployment | Performance tests, observability, backup and production deployment |

## Next release candidates

- Jira and Azure DevOps connectors
- Configurable organization templates
- Real-time collaborative review comments
- Qdrant indexing and semantic requirement reuse
- Background document jobs and notifications
- OIDC/SSO and enterprise identity provisioning
